Matlab函数1_4

简介

  1. 命令行edit可以直接新建文件
  2. 函数可以用来避免重复编写相同的功能
  3. 函数文件命名必须要和函数名同名

myRand.m

function myRand

a = 1 + rand(3, 4) * 9

end

输入与输出 I/O

  1. 函数运行会单独创建工作区, 在运行结束后会自动删除, 变量不会保存在界面中的工作区, 被称为是局部变量.

事实上就是作用域的问题

  1. 通过修改函数来获得输出
function a = myRand

a = 1 + rand(3, 4) * 9

end

然后需要命令行使用

b = myRand
  1. 修改传入参数
function a = myRand(low, high)
% 创建一个3*4的随机矩阵, 取值范围为[low,high]
a = low + rand(3, 4) * (high - low);

end
  1. 返回多个变量
function [a, s] = myRand(low, high)

a = low + rand(3, 4) * (high - low);

s = sum(a, "all");

% v = a(:); %给a中所有元素合并成一个列向量v(按列合并)

% s = sum(v); %计算列向量v中所有元素的和

end

函数的定义

exist函数可以查询是否是内置函数, 从而避免自定义函数与内置函数重名

子函数

可以在一个m文件中保存多个函数

只有文件中的第一个函数可以被其他程序调用, 其他函数被称为是子函数或者局部函数只能在文件内部被调用

作用域

可以设置全局变量从而使变量可以在函数外被操作

global a

注意到matlab中全局变量的声明及其赋值必须分开进行.

  1. 要尽量避免使用全局变量 这会导致引起一些难以诊断的错误

类似于命令行 edit 文件名 可以之间创建文件

  1. 1954 Fortran
  2. 1970s 开始现代语言 C C++ java C# 有利于大型程序

  4. Clever Moler 教授发明了Matlab
  5. 首先主要介绍矩阵和运算符1

变量的命名规则 

1. 只能使用字母数字以及下划线
2. 必须以字母开头
3. 命名长度有限制, 最多只能包含63个字符

clc 清理命令行

clear 清理工作区

命令行

  1. 语句结尾不打分号;或者只打逗号, matlab会自动echo
  2. 分号和逗号都可以使多个语句在一行执行
  3. 多行执行一个命令
